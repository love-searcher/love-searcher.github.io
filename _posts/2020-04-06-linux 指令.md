## linux：个人常用指令

#### 软件安装与卸载
yum update #更新系统
yum makecache #更新软件源缓存
yum search 软件名 #搜索软件
yum install 软件名 #安装软件
yum remove 软件名 #卸载软件
 

#### 软件的管理
service软件名 start #启动
service软件名 restart #重启
service软件名 stop #关闭
service软件名 status #状态
 

#### 文件与文件夹管理
touch 文件名 #创建文件
mkdir –p a/b/c/d #递归创建文件夹
cp a.a b.b #复制文件
mv a.a c.c #重命名或者复制
chmod 0755 d/ #目录权限
chown –R 用户名:用户组 e/ #目录归属
rm a.bc -rf #删除文件/文件夹
